Flask学习专栏

发表于 2020-05-11 | 分类于 Python | | 阅读次数:

Flask是由python实现的一个web微框架，“微”(micro) 并不表示你需要把整个 Web 应用塞进单个 Python 文件（虽然确实可以），也不意味着 Flask 在功能上有所欠缺。微框架中的“微”意味着 Flask 旨在保持核心简单且易扩展。Flask 不会替你作出过多选择，比如使用何种数据库。并且，Flask已经选择好的，比如使用何种模板引擎，是易于修改的。除此之外的一切都取决于你，因此 Flask 可以满足你的一切需求。
